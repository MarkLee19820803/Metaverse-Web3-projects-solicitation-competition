项目名称：Mage Tower（法师塔）

队内分工：

* KS Monika：策划，后端开发
* Yogo：设计，美工

* unicorn：合约

项目简介：法师塔是基于 ERC721 和 ERC20 的像素对战游戏，每个玩家通过选择不同颜色阵营，采集资源、占领地块、发起进攻。在结算日，占领地块最多的阵营会获得额外奖励。
